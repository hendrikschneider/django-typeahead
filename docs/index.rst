django-typeahead documentation
==============================

This `django widget <https://github.com/xvronny/django-typeahead>`__ contains Typeahead.js input field
with text suggestions functionality.

The widget implements `Typeahead.js <https://twitter.github.io/typeahead.js/>`__ to show typeahead fields in django
model forms and custom forms which can be configured easily for various input suggestions.
